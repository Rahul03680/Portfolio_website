Groot

An Android application which uses machine learning features to analyze the disease pattern on a plant leaf image and predict whether the plant is diseased or healthy.
If the plant is found diseased then some precautionary measures will be displayed to the user on the android application.
Technology used- Python, Keras, HTML,CSS, Flutter.
